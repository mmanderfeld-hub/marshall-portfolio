Pill Reminder System

Role: Project Developer

Status: Completed

Term: Fall 2025

Overview

Developed an Arduino-based pill reminder system using an LCD display and vibration alerts to assist with medication scheduling.

Technical Details

  • Arduino microcontroller
  • LCD interface
  • Vibration motor alerts
  • Time-based scheduling

Design Challenges

Ensuring reliable timing accuracy and intuitive user interaction within limited hardware constraints.

What I Learned

Strengthened skills in embedded programming, hardware integration, and user-centered system design.
