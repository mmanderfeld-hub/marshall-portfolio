Auto-Air Smart Fan

Role: Team Lead

Status: Completed — Awarded 1st Place in the Class

Term: Fall 2025

Overview

Auto-Air is a face-tracking smart fan that automatically follows the user. The fan utilizes a dynamic camera system mounted in the center of the fan. A Raspberry Pi, running OpenCV, uses the camera to detect faces and converts this data into vectors. An Arduino is sent this data, which it uses to determine the necessary adjustments for the servos it powers.

Technical Details

  • Raspberry Pi (OpenCV)
  • Arduino + Servo Control
  • 3D Printed Housing
  • Web Interface

Design Challenges

One major challenge was stabilizing face tracking to avoid oscillation and mechanical wear.

What I Learned

This project strengthened my skills in leading a small team, mechanical design, embedded systems, and system integration.
