Auto-Air Smart Fan

Status: Ongoing Development

Overview

Auto-Air is a face-tracking smart fan that automatically follows the user using computer vision and servo control.

Technical Details

  • Raspberry Pi (OpenCV)
  • Arduino + Servo Control
  • 3D Printed Housing
  • Web Interface

Design Challenges

One major challenge was stabilizing face tracking to avoid oscillation and mechanical wear.

What I Learned

This project strengthened my skills in mechanical design, embedded systems, and system integration.
